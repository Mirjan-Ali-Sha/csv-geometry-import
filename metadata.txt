about=This plugin allows you to import CSV files containing geometry data in various formats that QGIS doesn't natively support. Features include:
    - Support for WKT, WKB, EWKT, EWKB geometry formats
    - GeoJSON and JSON geometry parsing
    - KML geometry element parsing
    - Earth Engine (ee.Geometry) format support
    - TopoJSON geometry support
    - X-Y coordinate columns (latitude/longitude)
    - Auto-detection of geometry format
    - Custom CRS selection (default EPSG:4326)
    - CSV preview before import
    - Multiple encoding and delimiter options
